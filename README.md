# ☕ Chai Culture – Coming Soon Landing Page

A premium **Coming Soon Landing Page** designed for **Chai Culture**, inspired by the royal tradition and heritage of authentic Indian chai.

This landing page reflects the brand’s elegant identity using warm tones, golden accents, and a minimal luxury layout.

---

##  Features

 Royal and premium UI design  
 Brand logo + tagline included  
 Launching Soon announcement section  
 Email signup input with notify alert  
 Social media icons  
 Fully mobile responsive layout  
 Clean and well-structured code  

---

##  Design Choices

The page uses deep brown and gold tones to represent Indian royalty and warmth.  
Typography and spacing are kept elegant to match Chai Culture’s premium positioning while ensuring a smooth user experience across all devices.

- HTML5  
- CSS3  
- JavaScript
